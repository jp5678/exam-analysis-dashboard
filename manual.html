🏠 대시보드 개요
시험 성적 분석 대시보드는 Excel(XLSX) 파일 3종을 업로드하면 문항 분석, 성적 분포, 반별 비교, CQI 보고서 등을 자동으로 생성하는 웹 기반 분석 도구입니다. 별도 설치 없이 브라우저에서 바로 사용하며, 업로드된 데이터는 서버로 전송되지 않습니다.
🌐 웹 주소로 바로 접속 — 아래 주소를 브라우저(크롬·엣지·사파리 등)에 입력하면 설치 없이 즉시 사용할 수 있습니다.

📂 파일 업로드 안내
파일 1 문항분석표
문항별로 사전 계산된 통계 데이터(정답률, 변별도, 난이도 등)가 담긴 Excel 파일입니다.
항목내용
건너뛰는 행1행~5행 제외 → 6~7행이 헤더(머리글), 8행부터 실제 데이터
열 이름은 6행에서 읽으며, 7행처럼 숫자가 없는 머리글 연속 행은 자동으로 걸러집니다. 머리글이 6행 한 줄뿐인 파일도 그대로 사용 가능합니다.
필수 열문항번호, 정답률(또는 정답율)
선택 열변별도, 난이도, 평균, 표준편차
정답률 스케일0~1 또는 0~100 자동 감지
만점 자동 감지파일의 행 수(문항 수)로 시험 만점 자동 설정
예: 40문항 → 40점 만점, 성취기준 28점(환산 70점)
도구막대에서 수동으로 [적용]한 값이 있으면 수동 값이 우선됩니다.
💡 열 이름이 "정답률", "정답율", "Correct Rate", "CR" 등 다양한 형태여도 자동으로 인식합니다.
파일 2 성적분포도
0~100점 범위의 성적 분포 데이터입니다.
항목내용
건너뛰는 행1행~6행 제외 → 7행이 헤더(머리글)
형식 ①점수 열 + 인원수 열 (빈도표 형식)
형식 ②개인별 점수 목록 (1열에 점수만 있는 경우)
점수 범위0~100점 (환산점수 기준) 또는 원점수 자동 감지
파일 3 종합성적현황
반별 개인 성적이 담긴 다중 시트 Excel 파일입니다.
항목내용
Sheet1 (전체)1행~6행 제외 → 7행이 헤더 / 전체 학생 통합 데이터
반별 시트시트명 = 반 이름, 첫 행이 헤더
필수 열점수(원점수) 열
선택 열학번, 이름
⚠️ 전체 시트 인식: 시트명이 "Sheet1", "전체", "종합", "All", "Total" 중 하나여야 합니다. 나머지 시트는 모두 반별 데이터로 처리됩니다.
📤 업로드 방법
A
클릭 업로드 — 파일 카드 클릭 → 파일 선택 대화상자에서 Excel(.xlsx, .xls) 선택
B
드래그 & 드롭 — 파일을 카드 영역으로 직접 드래그하여 놓기
💡 3개 파일 중 1개 이상 업로드하면 성적 분석 시작 버튼이 활성화됩니다. 파일을 일부만 업로드해도 해당 파일 기반 분석이 가능합니다.
💡 파일 선택 대화상자에서 엑셀 파일이 희미하게 보여 선택되지 않던 문제가 개선되었습니다. 그래도 선택이 안 되면 파일을 카드 위로 드래그 & 드롭하세요.
📊 분석 탭 상세 안내
📋 문항분석 & 난이도 파일 1
문항별 정답률 막대차트(색상=난이도), 난이도 분포 파이차트, 정답률 분포, 난이도 분류 카드(매우 쉬움~매우 어려움), 난이도 순 정렬 차트, 문항 분석 상세표. 헤더 클릭으로 정렬 가능.
📊 성적분포 파일 2
7구간 히스토그램 (A+/A/B+/B/C+/C/미성취), 성취수준 파이차트, 통계 요약(평균·중앙값·SD·Q1/Q3·성취율), 누적 분포 곡선(CDF). 히스토그램 구간은 gradeOf()와 동일한 기준 적용.
🔴 오답 TOP 10 파일 1
정답률이 가장 낮은 10문항 자동 선별. 오답률, 변별도, 난이도 함께 표시. 집중 보완이 필요한 문항을 한눈에 파악 가능.
🎯 문항 변별도 파일 1
변별도 막대차트(우수/양호/보통/개선필요), 정답률 vs 변별도 산점도. 이상적 문항은 우상단(높은 정답률 + 높은 변별도).
⚖️ 반별 분석 파일 3
반별 평균·중앙값·표준편차 차트, 성취수준 분포, 통계 비교표, 미성취(환류) 학생 현황. 히스토그램 구간은 MAX_SCORE에 따라 자동 계산됨. 반 선택 버튼으로 필터링 가능.
🏆 성적 현황 파일 3
전체 또는 반별 개인 석차·학번·이름·원점수·환산점수·성취수준 표. 헤더 클릭으로 정렬. 반 선택 버튼으로 필터링 가능.
⚖️ 반 선택 필터 기능
  • 탭 상단 반 선택 버튼 클릭 → 해당 반의 통계로 요약 카드 갱신
  • 전체 버튼 → Sheet1(전체) 기준 통계로 복귀
  • 반별 분석 탭과 성적 현황 탭이 동시에 필터링됨
💡 데이터가 없는 탭은 흐리게 표시(dim)되어 클릭이 비활성화됩니다. 해당 파일을 업로드하면 자동으로 활성화됩니다.
💡 CQI 보고서 탭은 성적 분석 실행 전에는 안내 메시지만 표시됩니다. 파일 업로드 후 성적 분석 시작을 클릭해야 입력 양식이 나타납니다.

📐 문항 분석 주요 용어
정답률 (Correct Rate)
해당 문항에 정답을 맞춘 학생의 비율(0~100%). 낮을수록 어려운 문항입니다.
예: 정답률 60% → 10명 중 6명 정답
오답률
1 − 정답률. 정답을 맞추지 못한 학생의 비율입니다.
변별도 (Discrimination Index, D)
상위집단과 하위집단의 정답률 차이. 우수한 학생과 그렇지 않은 학생을 구분하는 능력입니다.
D ≥ 0.4 우수 D ≥ 0.3 양호 D ≥ 0.2 보통 D < 0.2 개선필요
난이도
오답률을 기준으로 5단계로 분류합니다.
매우 쉬움(오답<15%) 쉬움(15~29%) 보통(30~59%) 어려움(60~79%) 매우 어려움(80%↑)
표준편차 (SD)
점수의 분산 정도입니다. 클수록 학생 간 점수 격차가 큰 것을 의미합니다.
Q1 / Q3 (사분위수)
Q1: 하위 25% 경계점 / Q3: 상위 25% 경계점. Q3−Q1 = IQR(중간 50% 분포 범위)
🎯 성취율 · 환산점수 관련 용어
성취율
환산점수 70점 이상인 학생의 비율입니다.
예: 30명 중 24명이 70점 이상 → 성취율 80%
환산점수
공식: 환산점수 = 원점수 ÷ MAX_SCORE × 100
문항 수가 달라도 동일한 100점 기준으로 비교할 수 있습니다.
시험 만점 (MAX_SCORE)
파일 1(문항분석표)을 업로드하면 문항 수로 자동 설정되며, 도구막대에서 직접 입력·적용할 수도 있습니다(수동 값 우선).
예: 40문항 → 만점 40점, 성취기준 28점(환산 70점)
만점이 잘못 설정되면 환산점수·성취율이 왜곡되므로, 만점 초과 점수 경고 배너가 뜨면 반드시 확인하세요.
미성취(환류)
환산점수 70점 미만 학생으로, 학습성과 성취 기준에 미달한 경우입니다.
이 학생들을 대상으로 환류 교육(보충 수업, 개별 피드백 등)을 실시합니다.
CQI (Continuous Quality Improvement)
교과목 학습성과에 대한 지속적 질 개선 보고서입니다.
시험 결과를 분석하여 문제점을 파악하고, 다음 학기 교육의 질을 개선하기 위한 계획을 수립하는 체계적 문서입니다.
