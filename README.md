# 시험 성적 분석 대시보드

엑셀 성적 파일 3종을 업로드하면 문항 분석·성적 분포·반별 비교·CQI 보고서를 자동으로 생성하는 **웹 기반 분석 도구**입니다. 설치 없이 브라우저에서 바로 동작하며, 업로드한 데이터는 서버로 전송되지 않습니다.

> 청암대학교 간호학과 · 정종필 교수

## 주요 기능

- **문항 분석** — 문항별 정답률·난이도·변별도 차트, 오답 상위 10문항, 개선 필요 문항 자동 표시
- **성적 분석** — 성취수준(A+~미성취) 분포, 반별 비교, 미성취(환류 대상) 학생 명단
- **CQI 보고서** — 8개 섹션 자동 생성. ⑥문제점·⑦환류 계획·⑧개선 목표는 데이터 기반 초안 자동 작성 → 검토·수정
- **문서 출력** — PDF 저장(텍스트 검색 가능한 인쇄용 PDF 포함), CSV, 학생 1인 1페이지 개인별 리포트
- **학기 관리** — 세션 저장/복원, 중간·기말·학기 간 성취도 추이 비교
- **AI 심층 분석(선택)** — Claude API 키 등록 시 종합 총평·개선 제안 자동 작성 (학생 개인정보는 전송하지 않음)

## 사용 방법

1. [배포된 페이지](#배포)를 열거나, 저장소를 내려받아 `index.html`을 브라우저로 엽니다.
2. 엑셀 파일 3종(문항분석표·성적분포도·종합성적현황)을 업로드합니다.
3. **성적 분석 시작** 버튼을 누르면 분석 결과와 CQI 입력 양식이 표시됩니다.

자세한 사용법은 대시보드 우측 상단 **📖 메뉴얼** 버튼(또는 [manual.html](manual.html))을 참고하세요.

## 입력 파일 형식

| 파일 | 내용 | 머리글 위치 |
|------|------|-------------|
| 파일1 문항분석표 | 문항별 정답률·변별도·난이도·평균·표준편차 | 6~7행 (8행부터 데이터) |
| 파일2 성적분포도 | 점수별 인원수 또는 개인 점수 | 7행 |
| 파일3 종합성적현황 | 반별 개인 성적 (시트명 = 반 이름) | 전체 시트 7행 / 반별 시트 1행 |

## 기술 스택

순수 정적 웹 (빌드 도구 불필요). 라이브러리는 `libs/`에 로컬 번들되어 **오프라인에서도 동작**하며, 로드 실패 시 CDN으로 자동 폴백합니다.

- [SheetJS(xlsx)](https://sheetjs.com/) — 엑셀 파싱
- [Chart.js](https://www.chartjs.org/) — 차트
- [html2canvas](https://html2canvas.hertzen.com/) + [jsPDF](https://github.com/parallax/jsPDF) — PDF 내보내기

## 파일 구조

```
index.html        대시보드 (구조)
manual.html       사용 매뉴얼
styles.css        스타일
js/
  core.js         상수·상태·통계·유틸
  parse.js        파일 업로드·엑셀 파싱
  render.js       분석 실행·차트·표 렌더링
  export.js       PNG·CSV·PDF·인쇄·개인별 리포트
  report.js       AI 분석·CQI 보고서
  session.js      세션 저장·복원·시험 비교
libs/             로컬 번들 라이브러리
```

## 개인정보 보호

모든 분석은 사용 중인 브라우저 안에서만 이루어집니다. 업로드한 성적 파일과 학생 정보는 외부로 전송되지 않으며, 인터넷 연결 없이도 사용할 수 있습니다. (AI 심층 분석 기능만 선택적으로 인터넷을 사용하며, 이때도 통계 요약만 전송되고 학생 이름·학번은 전송하지 않습니다.)
